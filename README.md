# 🎵 Cotton Candy Music Player

A visually delightful, lightweight, and customizable web-based music player built with HTML, CSS, and JavaScript. Inspired by soft pastel gradients, it lets you play songs from a built-in playlist or upload your own!

---

## 🌟 Features

- 🎶 Play, pause, skip to next/previous songs
- 📜 Built-in playlist support
- 📤 Upload your own audio files
- 🎧 Volume and progress control
- 🎨 Elegant, responsive "cotton candy" UI theme

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, gradients, transitions)
- JavaScript (Vanilla)
